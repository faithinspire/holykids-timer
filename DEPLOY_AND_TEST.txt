3. Click "Setup" under Biometric
4. Scan fingerprint
5. WATCH CONSOLE - any errors?
6. Check if status shows "✅ Enrolled"

┌────────────────────────────────────────┐
│ STEP 5: CHECK DATA (30 seconds)        │
└────────────────────────────────────────┘

In console, type:
JSON.parse(localStorage.getItem('holykids_staff'))

Look for:
✓ biometric_enrolled: true
✓ biometric_raw_id: [1,2,3,4,5...]

┌────────────────────────────────────────┐
│ STEP 6: TEST CLOCK-IN (1 minute)       │
└────────────────────────────────────────┘

1. Go to Clock In page
2. Click "Scan Fingerprint"
3. WATCH CONSOLE - should see:
   - "All staff: [...]"
   - "Staff Name: enrolled=true, has_raw_id=true"
   - "Enrolled staff count: 1"
4. Scan fingerprint
5. Should see success screen
