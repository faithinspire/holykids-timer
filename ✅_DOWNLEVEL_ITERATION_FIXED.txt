==========================================
WHAT WAS THE ERROR:
==========================================

Vercel build failed with:
"Type 'Set<string>' can only be iterated through when using 
the '--downlevelIteration' flag or with a '--target' of 
'es2015' or higher."

This happened on line 309 in app/admin/reports/page.tsx:
const departments = [...new Set(staffList.map(s => s.department))]

The spread operator [...] on a Set requires downlevelIteration
when targeting ES5.

==========================================
WHAT WAS FIXED:
==========================================

Added to tsconfig.json:
"downlevelIteration": true

This allows TypeScript to properly compile Set/Map spread
operators when targeting ES5.
